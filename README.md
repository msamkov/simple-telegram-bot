# Простой бот telegram

Установить:  
* sudo -H pip install pytelegrambotapi 
* sudo -H pip install configparser

Создать файл token.ini:  
[TOKEN]  
TOKEN = ваш token без ковычек

Запуск:  
  python simple_telegram_bot.py
